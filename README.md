# TGSTurbo

Fast telegram spammer for you

## Getting Started

### Installing

#### Fast install

```
curl https://raw.githubusercontent.com/liriondev/TGSTurbo/main/install.sh | bash
```

#### Go to directory

```
cd TGSTurbo
```

## Running

```
pipenv run python3 main.py
```
